# Candlestick chart
Personal Vue project. Shows a candlestick chart for any stock using the Alpha Vantage API and ApexCharts librairie.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
